TO BASE64
=========
[![Build Status](https://travis-ci.org/ArvoGuo/tobase64.svg?branch=master)](https://travis-ci.org/ArvoGuo/tobase64)


##anykind of image or font file

##convert to base 64

image: [.png, .jpg....]

font:  [.eot, .svg, .ttf, .woff....]



##install

npm i -g tobase64

##usage

tobase64 ./xxx/xxx.png
